# Sample User Story

## User Story

As a registered user,
I want to reset my password using my email address,
So that I can regain access to my account if I forget my password.

## Acceptance Criteria

1. The user can request a password reset by entering a registered email address.
2. The system displays a generic confirmation message after the request is submitted.
3. The system does not reveal whether the email address exists.
4. The reset link expires after a defined period.
5. The new password must comply with the password policy.
6. The user receives a clear error message if the new password is invalid.
7. The user can log in with the new password after a successful reset.
